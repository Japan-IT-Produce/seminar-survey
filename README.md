# セミナー リアルタイム集計アンケート

セミナー会場で参加者がスマートフォンからQRコードを読み取って回答し、その結果をリアルタイムでプロジェクタースクリーンに投影できるWebアプリケーションです。管理者はダッシュボードからセッションの管理と結果の確認が行えます。

## 主な機能

- QRコードによる簡単な参加者アクセス
- スマートフォン最適化された回答画面（5問のアンケート）
- Socket.IOによるリアルタイム集計・結果反映
- プロジェクター投影用のダークテーマ結果画面
- 管理者ダッシュボードでのセッション管理・Chart.jsグラフ表示
- 回答受付の開始・終了・リセットをリアルタイムに制御

## セットアップ

```bash
cd seminar-survey
npm install
npm start
```

サーバーが起動したら http://localhost:3000 にアクセスできます。

開発時は `npm run dev` でファイル変更時に自動再起動します。

## 使い方

1. **セッション作成**: ブラウザで http://localhost:3000/admin.html を開き、セッション名を入力して作成します。
2. **QRコード表示**: 「投影画面を開く」ボタンでプロジェクター用画面を表示し、QRコードを会場に投影します。
3. **回答受付開始**: 管理画面の「回答受付開始」ボタンを押すと、参加者が回答できるようになります。
4. **参加者回答**: 参加者はスマートフォンでQRコードを読み取り、アンケートに回答します。
5. **結果確認**: 回答はリアルタイムで管理画面と投影画面のグラフに反映されます。
6. **終了**: 「回答受付終了」ボタンで受付を締め切ります。必要に応じて結果をリセットできます。

## 技術構成

- **バックエンド**: Node.js, Express, Socket.IO
- **フロントエンド**: Vanilla HTML/CSS/JavaScript
- **グラフ描画**: Chart.js
- **QRコード生成**: qrcode (サーバーサイド)
- **リアルタイム通信**: Socket.IO
